# Matimato 2.6.0

## Summary

Release 2.6.0 delivers the market-research retention and teachability package:

- Seasonal collection album and deterministic reward claims.
- Always-available rules help with contextual match hints.
- Solo AI opponent profiles with difficulty/personality selection.

## Verification

- `npm run lint`
- `npm test`
- `npm run build`
- `npm run mobile:smoke`

## Rollback

- Set `NEXT_PUBLIC_MATIMATO_SEASONAL_EVENTS=false` to hide the album and reward track.
- Set `MATIMATO_SEASONAL_EVENTS_ENABLED=false` to pause new server-side season progress.
- Set `NEXT_PUBLIC_MATIMATO_RULE_ASSIST=false` to hide persistent help buttons.
- Set `NEXT_PUBLIC_MATIMATO_AI_PROFILES=false` to keep solo play on the default rookie AI behavior.
- Use Vercel rollback for broader web/PWA regressions.
