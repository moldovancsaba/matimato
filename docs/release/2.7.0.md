# Matimato 2.7.0

## Scope

- Delivered #65: relationship-safe Friends screen, mutual friend creation from lobby/recap/profile invite links, idempotent daily 15 XP gifts, block/remove controls, friend battle entry through the normal V2 lobby, telemetry, tests, docs, and rollback flags.
- Delivered #67: public-safe `/replay/[id]` route and `/api/replays/[id]` DTO, sanitized move playback, summary-only fallback for old matches, replay share/copy and conversion actions, telemetry, tests, docs, and rollback flags.

## Verification

- `npm test`
- `npm run lint`
- `npm run build`
- `npm run verify`
- `npm audit --omit=dev`
- Vercel deployment verification through `/api/health`

## Rollback

- Friends: set `NEXT_PUBLIC_MATIMATO_FRIENDS=false` and `MATIMATO_FRIENDS_ENABLED=false`.
- Replays: set `NEXT_PUBLIC_MATIMATO_REPLAYS=false` and `MATIMATO_REPLAYS_ENABLED=false`.
- Stored friendships, gift ledgers, match snapshots, and move logs are preserved for recovery.
