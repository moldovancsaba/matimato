# Matimato 2.5.0

## Summary

Release 2.5.0 adds the iOS mobile app delivery foundation: installable PWA metadata, offline recovery shell, Capacitor iOS project, mobile viewport regression harness, runtime telemetry, and App Store/TestFlight readiness documentation.

## Verification

- `npm run lint`
- `npm run verify`
- `npx cap sync ios`

The local unsigned simulator build is blocked until full Xcode is selected instead of Command Line Tools. See `docs/ios-mobile.md`.

## Rollback

- Use Vercel rollback for web/PWA failures.
- Set `NEXT_PUBLIC_MATIMATO_SERVICE_WORKER=false` if service worker registration needs to stop.
- Stop or expire native TestFlight builds if a wrapper regression is discovered after Apple distribution begins.
